## Main Project

### Carefull - AI 스마트 복약 관리 서비스

보호자와 환자, 복약 디바이스를 연결하여  
복약 일정 관리, 복약 이력 기록, 보호자 알림을 제공하는 스마트 복약 관리 서비스입니다.

#### 담당 역할

- Node.js + Express 기반 REST API 서버 구현
- PostgreSQL 연동 및 SQL 기반 데이터 처리
- 소셜 로그인 OAuth 흐름 및 JWT 인증 처리
- 보호자-환자-디바이스 연결 API 구현
- 복약 일정과 복약 실행 기록 분리 처리
- FCM 기반 보호자 알림 흐름 구현
- ElevenLabs 기반 보호자 음성 알림 처리 흐름 구현
- React 프론트엔드와 API 응답 구조 연동

#### 핵심 구현 경험

| Area | Experience |
|---|---|
| Auth | 소셜 로그인 후 JWT 발급 및 인증 API 처리 |
| User Flow | 로그인 사용자 기준 환자/기기 등록 흐름 구현 |
| Medication | 복약 일정 등록, 조회, 삭제 처리 |
| Dashboard | 복약 기록 기반 대시보드 데이터 제공 |
| Notification | FCM 토큰 등록 및 보호자 알림 전송 흐름 구현 |
| Environment | 환경변수 기반 개발/배포 환경 분리 |
